# Contributing to Awesome Voice AI

## Adding a Tool

### Option 1: Open an Issue
Use the [Submit a Tool](https://github.com/ALLBOTSIO/awesome-voice-ai/issues/new?template=submit-tool.yml) template.

### Option 2: Pull Request
1. Fork this repo
2. Add the tool to the appropriate section in README.md
3. Follow the existing table format
4. Submit a PR

## Guidelines
- Tool must be directly related to voice AI
- Include a factual one-line description (no marketing language)
- Open source tools: include star count and license
- Commercial tools: include key differentiators
- No affiliate links
- Alphabetical order within subsections (open source vs commercial)
